🔧 Setup Instructions

If Ollama is not detected:

  1. Install Ollama: brew install ollama
  2. Pull a model: ollama pull llama3.1:8b
  3. Start with CORS: OLLAMA_ORIGINS=* ollama serve
  4. Refresh this page
