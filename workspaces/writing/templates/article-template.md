## SEO & Formatting Checklist

- [ ] Headline includes primary keyword naturally
- [ ] Subheadings use H2 and H3 tags
- [ ] Introduction includes the target keyword
- [ ] Article is scannable (lists, short paragraphs, bold text)
- [ ] Includes at least one image description or diagram placeholder
- [ ] Meta description written (150-160 characters)
- [ ] URL slug is descriptive and keyword-rich
- [ ] Internal links to related content included
- [ ] External links to authoritative sources included

---

## Tone & Style Notes

**Neutral Style (Default for articles):**
- Conversational but informative
- Short paragraphs (2-4 sentences)
- Subheadings every 200-300 words
- Second person ("you") for direct address

**Technical Style:**
- Code blocks and technical precision
- Assume semi-technical audience
- Link to documentation

**Academic Style:**
- Formal tone with citations
- Abstract or executive summary
- References section
